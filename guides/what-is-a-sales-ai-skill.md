# What Is a Sales AI Skill?

A sales AI skill is a set of instructions that teaches an AI assistant how to handle one repeatable sales task.

Instead of explaining the same process every time, you give the assistant a tested method to follow. The skill can define what information to use, what the output should contain and what the salesperson must still check.

## A Workflow and a Skill Are Different

A workflow explains the process to a person.

A skill gives the process to an AI assistant in a form it can follow repeatedly.

The person still owns the judgement, customer relationship and external action.

## What Skills Are Good For

Skills can help with work such as:

- Turning call notes into consistent evidence
- Separating facts from assumptions
- Preparing a first draft
- Spotting missing information
- Applying the same checks every time
- Learning from repeated errors

They are most useful when the task happens regularly and a good result has a recognisable shape.

## What Skills Cannot Decide Safely

A skill should not decide whether you have enough evidence to make a promise, change an opportunity stage or contact a customer without review.

It should not:

- Invent facts, dates or commitments
- Send customer messages automatically
- Change CRM records without approval
- Decide that a prospect is qualified based on weak evidence
- Replace your company's policies
- Use information you are not allowed to process

## How to Adapt a Skill

Start with the business problem, not the AI tool.

Ask:

1. What task are we trying to improve?
2. What information is genuinely needed?
3. What should the output contain?
4. Which decisions still belong to a person?
5. What would make the result unsafe or misleading?
6. How will we know whether it is better than the current process?

Then adapt the terminology, fields and checks to your own sales process. Keep company policies, customer information and system access in approved environments.

## Test Before Using It on Real Work

Start with a fictional example that contains deliberate traps, such as an estimated number, a conditional commitment or an unconfirmed meeting.

Check whether the skill:

- Preserves uncertainty
- Identifies missing information
- Avoids inventing momentum
- Produces something useful
- Makes the human checks obvious

Improve the instructions when the same mistake appears more than once.

## Before Using a Sales AI Skill

- Identify the business problem
- Check which tools are approved
- Use the minimum necessary information
- Keep facts and assumptions separate
- Review the proposed action
- Keep customer communication and CRM changes under human control
- Measure whether the workflow improves the result
