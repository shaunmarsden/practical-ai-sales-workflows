## Consistent Post-Call Documentation for Bramfield's Claims Handling Team

### Overview

Bramfield Insurance Group's thirty claims handlers currently document policyholder calls with varying levels of consistency, which matters more than usual here because claims documentation sits under FCA record-keeping obligations. A three-week pilot with twelve claims handlers tested a consistent process for turning a claims call into a summary, a suggested next-action list and a draft case file update, with every output reviewed by the handler before anything is saved.

### The Pilot Result

Twelve claims handlers ran the process against real policyholder calls over three weeks. Three changes were observed:

**Call wrap-up time.** Before the pilot, wrapping up a call, notes, next actions, case file update, was taking handlers close to eighteen minutes on average. Timed across the three weeks rather than estimated, that came down to around seven minutes.

**Next-action guidance quality.** The process's suggested next actions matched what an experienced handler would do on the same call, reducing the judgement gap between newer and more experienced handlers.

**Case file draft quality.** Suggested case file updates needed light checking rather than a full rewrite before a handler saved them.

### What Changes With a Full Rollout

Extending the pilot from twelve claims handlers to all thirty would bring the same consistent process to the whole team, rather than leaving it as an approach twelve handlers happened to trial. Every handler, regardless of experience, would work from the same structure for documenting a call.

### Commercial Proposal

Two pricing paths were discussed, and the difference matters:

- **With a two-year term signed now:** year one is £52 per seat per month for all thirty seats (£18,720 for the year), and year two drops to £45 per seat per month for the same thirty seats (£16,200 for the year). Two-year total: £34,920. The £45 rate is only available as part of this two-year commitment.
- **Without a two-year commitment:** £52 per seat per month applies in both years, with no reduction in year two. Two-year total on that basis: £37,440.

These figures were confirmed on the pilot review call. The reduced year-two rate should not be presented as available on its own; it depends on the two-year term being agreed at the same time as year one.

### Data and Compliance

Bramfield's call recordings are already retained for seven years under existing FCA obligations. Outputs generated by this process, summaries and suggested next actions, stay inside Bramfield's own systems and inherit the same retention and access policy already in place, since nothing leaves Bramfield's environment. Written confirmation of this from the vendor's compliance team has been requested and is in progress; it has not yet been received, and it should be treated as outstanding until Bramfield has it in writing, not assumed to be already settled.

### Time Commitment

No change to a handler's day-to-day workload beyond what the pilot already used. Rolling out to the remaining eighteen claims handlers would need a single onboarding session per person to review the process.
